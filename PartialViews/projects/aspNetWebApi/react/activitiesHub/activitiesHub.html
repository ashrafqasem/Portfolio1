Activities Hub

Your Passion. Our Platform.

Activities Hub is a software program that enables users to easily store and find contact information, such as names, addresses, and telephone numbers. They are contact-centric databases that provide a fully integrated approach to tracking all information. Hub is a one of its kind event learning and hobby experience space where people from all age groups learn, share experience new skills for personal and professional use. With trained professionals as their flag bearers, Activity Hub ensures that everyone who joins us learns skills that are cherished for their entire life. Like Us

A role based security system was implemented to create permission based roles. The role of Admin, Super User and User restrict what the user can see and do.

The system was built with ASP.NET Core MVC and C#. The Fron-End was developet with HTML, CSS, Bootstrap, DataTables, JavaScript, and JQuery. The Back-End uttilizes MVC, C#, and Entity Framework. The Database wase built with SQL Server and hosted on Azure and AppHarbor using Git/ GitHub. Please visit the site and try it out. »
